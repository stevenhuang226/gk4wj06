# 使用說明
- 需要提供一個log檔案作為log紀錄使用（可選）
- 必須提供一個SQLite3可以使用的檔案，作為SQLite3資料庫使用
- 必須在執行目錄下，有一個.env檔案，內容為 DC_TOKEN=你的discord機器人token
